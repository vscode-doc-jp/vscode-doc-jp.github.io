VS Code で HTML プログラミング

Last updated: 2017-02-27

VS Code は HTML プログラミングの基本的なサポートを提供します。拡張機能をインストールすることでより多くの機能性を得ることができます。

(編集メモ: 本家でここに人気拡張4つが表示

Tip: 上記の拡張機能は動的に照会します。説明とレビューを読んで、最適な拡張機能を決定するために拡張機能のタイルをクリックしてください。そうして詳細を Marketplace で参照してください。

IntelliSense

HTML を入力するとき HTML IntelliSense を介して候補を提供します。下の画像では、HTML閉じタグ要素 </ div> と候補された、要素の固有コンテキストリストを表示します。

また 要素, タグ, HTML 5, Ionic および AngularJS タグのための候補を提供しています。

埋め込み CSS と JavaScript でも動作します。しかし、(ノート) (それは) スクリプト と スタイル 含む(s) は(s) フォローされない。(However, note that script and style includes are not followed,)言語サポートは HTML の内容だけをサポートします。

(編集メモ: よくわからない

⌃Space (Windows, Linux Ctrl+Space) を押すことで IntelliSense をいつでも起動できます。

ビルトインコード補完するかどうかを構成できます。候補を表示したくない場合は、 ユーザーまたはワークスペース設定 で次を設定します。

ホバー

HTML タグまたは埋め込まれたスタイル,JavaScript の上にマウスを移動すると、カーソルの下にシンボルの詳細を表示します。

検証

HTML 言語サポートは、埋め込まれたすべてのJavaScriptとCSSの検証を実行します。

HTML フォーマット

HTML ソースコードの書式を整えるために ⌘K ⌘F (Windows, Linux Ctrl+K Ctrl+F) を押して選択範囲をフォーマットすることができます。

Tip: フォーマッタは html.format.unformatted 設定に登録しているタグをフォーマットしません。埋め込み JavaScrirpt は script タグが除外していない限り、フォーマットします。

Tip:`User and Workspace Settings で HTML フォーマッタの設定を構成できます。

Emmet

Emmet は Tab を押すだけで機能します。

Tip: 有効な略語については Emmet cheat sheet の HTML セクションを参照してください。

HTML Emmet の略語を他の言語で使用したい場合は Emeet 構文プロファイル(html, cssなど)を emmet.syntaxProfiles 設定に関連付けることでできます。設定は language id を使用して Emmet プロファイルに関連付けます。
