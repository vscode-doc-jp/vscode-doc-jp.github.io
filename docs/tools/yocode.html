Yo Code - 拡張機能ジェネレータ

Last updated: 2017-03-14

(編集メモ: 英文の特性上、箇条書きを直訳すると上の文の説明を後からすることになってわかりづらい

私達はYeoman generatorを提供し、拡張機能の公開手順を提供しています。

Generatorのインストール

YeomanとVS Code拡張ジェネレータをコマンドプロントから入手します:

Yo Codeの実行

Yeoman ジェネレータはステップ形式で、入力する情報と拡張機能を作成する方法を説明してくれます。

コマンドプロントで次を実行して、ジェネレータを起動します:

ジェネレータオプション

ジェネレータは新しい拡張機能のひな形を作成もしくは、既存のTextMate定義ファイルに基づいて言語、テーマ、スニペットに使用する拡張機能を作成できます。

New Extension in TypeScript

“hello world”コマンドを実装する拡張機能を作成します。入門として作成してみてください。

このコマンドで次のことが行われます

  • 拡張機能の識別子を求められます。またこの名前で現在のディレクトリにフォルダを作成します。
  • ソースフォルダ、テストフォルダ、出力フォルダを含むディレクトリ構築します。
  • package.jsonと拡張機能のメインファイルをテンプレートとして出力します。
  • launch.jsonとtasks.jsonを設定して、F5で拡張機能をデバッグしたり、デバッガでアタッチしたりできるようにします。
  • オプションでGitリポジトリを設定します。

作成したフォルダでVS Codeを開きます。またこのフォルダには、次の手順のクイックガイドvsc-extension-quickstart.mdファイルを含んでいます。なおこの拡張機能は、拡張API用のIntelliSenseが機能するようにセットアップ済みです。

New Extension in JavaScript

JavaScriptでも上のTypeScriptと方法は同じです。なおこの拡張機能は、拡張API用のIntelliSenseが機能するようにセットアップ済みです。

New Color Theme

TextMateに基づいて新しい配色テーマを提供する拡張機能を作成します。

このコマンドで次のことが行われます

  • TextMate(.tmTheme)の場所を URL かファイルパスで入力するように求められます。このファイルは作成した拡張機能にインポートします。
  • テーマ名とベーステーマ(light or dark)が求められます。
  • 拡張機能の識別子を求められます。またこの名前で現在のディレクトリにフォルダを作成します。

作成したフォルダでVS Codeを開き、拡張機能を実行して新しいテーマをテストします。
また、次の手順のクイックガイドvsc-extension-quickstart.mdをチェックしてください。

New Language Support

新しい言語カラライザーを提供する拡張機能を作成します。

このコマンドで次のことが行われます

  • TextMate(.tmLanguage, .plist or .json)の場所を URL かファイルパスで入力するように求められます。このファイルは作成した拡張機能にインポートします。新しい文法を作成する場合は、空のままにしてこれをスキップします。

  • 拡張機能の識別子を求められます。またこの名前で現在のディレクトリにフォルダを作成します。

作成したフォルダでVS Codeを開き、拡張機能を実行して新しいカラライザーをテストします。また、次の手順のクイックガイドvsc-extension-quickstart.mdをチェックしてください。作成した定義済み言語構成ファイルで、言語が使用するコメントやブラケットのスタイルを確認します。

New Code Snippets

新しいコードスニペットを提供する拡張機能を作成します。

このコマンドで次のことが行われます

  • TextMateスニペット(.tmSnippet)かSublimeスニペット(.sublime-snippet)の場所を URL かファイルパスで入力するように求められます。これらは、VS Codeスニペットファイルに変換されます。
  • スニペットを適用する言語を求められます。
  • 拡張機能の識別子を求められます。またこの名前で現在のディレクトリにフォルダを作成します。

作成したフォルダでVS Codeを開き、拡張機能を実行して新しいスニペットをテストします。また、次の手順のクイックガイドvsc-extension-quickstart.mdをチェックしてください。

拡張機能の読み込み

拡張記事を読み込むには、ファイルを VS Code extension フォルダにコピーする必要があります。Installing Extensionsで詳細を確認してください。
