翻訳メモ: 単語

  • bracket matching : 括弧強調
  • suggestion : 提案と候補の両方の意味で使われていて使い分けが難しい
  • completion : 入力候補 入力補完 補完の意味で使われていて使い分けが難しい
  • credential management: git用語 認証情報
  • credential prompts: ms用語 資格情報プロンプト
  • shoud > may want to : するべきです > 必要がある
  • closure : 閉じタグ 終了タグ 前者を採用
  • blazing fast : 極めて速い
  • build asset : MS 用語としても IT 用語としても出てこないし、用語として出てこない 意味としては「依存関係」に当たりそうだがよくわからない
  • colorizer : 色付けの意味を含むが固有名詞的に訳した方がわかりやすそうなので カラライザー で統一
  • Ubuntu based distributions : Ubuntu ベースのディストリビューション ベースを入れなくても意味が伝わるので Unbuntu ディストリビューション
  • graphical software center : Ubuntu にさほど詳しくないが グラフィカル ソフトウェア センターなんていうものを見つけることができなかった、ソフトウェア センターで訳す
  • ENOSPC(No space left on device) Error : ディスクに空き領域がないことを示すエラー
    • inotify : ァイルシステムイベントを監視する仕組み これの上限を 監視対象数上限
  • The character in the font : character font とは違う? フォントの中の文字とは(哲学)
  • glyph : グリフ 絵文字
  • Extension Marketplace : Marketplace だけの時もあるが違いがないので、Marketplaceで統一
  • tokenizer : トークナイザ 関数などで、入力文字列をトークンに分解するもの
  • empty string : 空文字列
  • one-index based : 添字が1から始まるインデックス方式 0オリジン (https://msdn.microsoft.com/ja-jp/library/ms233801.aspx)
  • regular expression replacements : 正規表現での置換
  • interpolated shell code : ???
  • transformations : 変換
  • extension manifest : 拡張機能マニフェスト
  • Editing Evolved : 進化した編集 (UI)
  • Interactive : 「インタラクティブな」が一般だと思っていたが UI 翻訳で「対話型」になったどうしよう
  • Conditional Breakpoints : 条件付ブレークポイント 条件付きブレークポイント どちらでもいいが後者
  • semantic analysis : 意味解析
  • single-file : 単一ファイル
  • Peek Definition : 定義をここに表示
  • Peek : ピーク
  • peek editor : ピークエディタ
  • skeleton : スケルトン -> ひな形
  • Problems Panel : モデルビュー Pnelをビューに訳すのが正しいらしいが気がついたのが遅かった
  • 編集途中で変えた内容
    • navigation : ナビゲーションで訳してきたけどよく考えれば移動の意味で使う方が早い
    • item : UIの翻訳を見てもアイテムと訳すより項目と訳した方が早い
