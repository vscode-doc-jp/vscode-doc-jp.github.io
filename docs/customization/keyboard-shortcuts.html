Keyboard shortcuts

Last updated: 2017-02-27

本家サイトにはURLが張られていませんが、日本語訳では残しておきます。

ver : 1.9.1

General

KeyCommand
⇧⌘P (Windows, Linux Ctrl+Shift+P), F1コマンドパレット
⌘P (Windows, Linux Ctrl+P)ファイルに移動する
⇧⌘N (Windows, Linux Ctrl+Shift+N)新しいウィンドウ/インスタンス
⌘W (Windows Ctrl+Shift+W, Linux Ctrl+W)ウィンドウ/インスタンスを閉じる

Home行頭に移動

⌃PageDown (Windows, Linux Ctrl+Down)下にスクロール(行)
